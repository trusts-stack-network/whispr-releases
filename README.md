# Whispr — Anonymous P2P Social Network

End-to-end encrypted messaging, calls, and crypto payments. No servers, no tracking, no identity required.

Built on [Trust Stack Network](https://tsnchain.com) — post-quantum secure.

## Download

See [Releases](https://github.com/trusts-stack-network/whispr-releases/releases) for the latest binaries.

| Platform | Download |
|----------|----------|
| Linux | `whispr-node-linux-amd64` |
| Windows | `whispr-node-windows-amd64.exe` |
| macOS | Coming soon |

## Features

- P2P encrypted messaging (no central server)
- Voice/video calls (WebRTC, E2E encrypted)
- Built-in TSN crypto payments
- AI assistant (local, private)
- Post-quantum cryptography (ML-DSA-65)
